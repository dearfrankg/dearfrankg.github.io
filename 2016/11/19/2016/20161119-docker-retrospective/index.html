    Time to evaluate my docker training

    Training Curriculum

    Getting Started:

    Use docker CLI to:

    • images: list, run, tag and build images
    • containers: list-running, list-all, start, stop, restart
    • docker hub: create account, find, pull and push images

    Learn By Example:

    Use docker CLI to:

    run apps

    • containers: run ephemeral, interactive and daemonized containers
    • containers: show logs, start, stop, restart and remove containers

    work with containers

    • docker: show client/server versions, use docker help
    • containers: start, stop, restart and remove containers
    • containers: assign name
    • containers: get ports, logs, processes
    • containers: inspect, inspect range

    Updating and committing an image

    Goal: clone nginx image and edit it to show new html page

    • images: search, list, pull, tag
    • images: run interactively to edit image
    • images: use ctrl-p, ctrl-q to leave without the exit command
    • images: get containerId
    • images: commit to exiting or new image
    • images: get imageId and compare imageIds

    Building an image from a Dockerfile

    • build image: create project folder
    • build image: create Dockerfile by entering Dockerfile commands
    • build image: build image
    • build image: tag image
    • build image: show image digests
    • build image: push to docker hub
    • build image: remove local image

    network containers

    • network containers: list networks, create network, remove network
    • network containers: add container to network, disconnect container from network
    • inspect network: inspect network
    • inspect container: inspect container, inspect container range

    data volumes

    • data volume: add anonymous volume, locate volume
    • data volume: mount host directory as volume, mount shared-storage as volume
    • data volume: mount host file as volume
    • data volume: create volume, using :z or :Z to relabel volumes

    data volume containers

    • data volume containers: create a data volume container
    • data volume containers: mount the data volume container in another container
    • data volume containers: find and remove dangling volumes

    backup data volumes

    • backup data volumes: backup, restore or migrate data volumes

    remove anonymous volumes

    • remove anonymous volumes: locate and remove anonymous volumes

    store images on Docker Hub

    • Docker Hub: login, search, pull, push
    • Docker Hub: private repos, teams, automated builds, web-hooks
    • Docker Hub: setup automated builds

    Training Curriculum Summary

    Getting Started:

    • a taste of images, containers and docker hub

    Learn By Example:

    • running apps
    • working with containers
    • building and updating images
    • networking containers
    • inspecting networks and containers
    • working with volumes and volume containers
    • working with docker hub
